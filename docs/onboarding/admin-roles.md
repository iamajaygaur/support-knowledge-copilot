# Admin Roles Guide

## Role levels

| Role | Capabilities |
| --- | --- |
| `member` | Use product features, create tickets |
| `team_lead` | Assign tickets, view team analytics |
| `admin` | Manage users, SSO, API tokens |
| `owner` | Billing, contract, and destructive account actions |

## Granting admin

Admin role requests require IT Security approval and are completed within **1 business day**. Owners can only be changed by Finance + Security jointly.

## Least privilege

Support agents should use `team_lead` unless they need user-security actions such as MFA reset, which require `admin`.
