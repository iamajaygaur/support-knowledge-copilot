# Notifications FAQ

## Email notifications

Account and security emails are sent from `noreply@example.com`. Users should allowlist this address if messages land in spam.

## Slack notifications

Workspace admins can enable Slack alerts under **Settings → Notifications → Slack**. Alerts support ticket assignment, SLA breach warnings, and failed sync jobs.

## Quiet hours

Quiet hours suppress non-urgent notifications between **22:00 and 07:00** local time. Urgent priority tickets always notify immediately.
