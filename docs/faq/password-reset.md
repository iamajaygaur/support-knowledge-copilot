# Password Reset FAQ

## How do employees reset their password?

Employees can reset passwords from the login page using **Forgot password**. A reset link is sent to the verified work email and expires in **30 minutes**.

## Temporary passwords

Support agents may issue a temporary password from the Admin Console under **Users → Security → Issue temporary password**. Temporary passwords expire after **24 hours** and require a change on first login.

## Common failures

If the reset email is not received, confirm the user is not in **SSO-only** mode. SSO-only accounts must reset credentials through the identity provider, not through the product password form.
